# CPP 3D Renderer

Currently working on a readme, please note this is an amature project in progress.

## Dependencies
- OpenGL
- GLUT / freeglut
- C++23

## Building
\```bash
mkdir build && cd build
cmake ..
make
./Renderer3D
\```

## Controls
| Input | Action |
|-------|--------|
| Left drag (empty space) | Rotates camera |
| Left drag (particle) | Grabs and throws the particle |
